## Overview

**system-prompts-and-models-of-ai-tools** is an open-source GitHub repository that aggregates system prompts, internal tools, and AI model details from over 25 popular AI coding assistants, including Cursor, Devin, and Replit. It serves as a comprehensive reference for developers and AI enthusiasts who want to understand the underlying prompts and architectures powering modern AI coding tools. With over 138k stars, it’s a highly visible community-driven project licensed under GPL-3.0.

**awesome-claude-skills** is a curated GitHub repository focused specifically on Claude AI skills, resources, and tools. It targets developers and AI enthusiasts looking to customize and extend Claude AI workflows, covering automation, agent skills, MCP (Model Context Protocol), and workflow automation. It’s also open-source and community-driven, but with a narrower, more specialized scope.

Both tools are free and open-source, but they serve different purposes: one is a broad reference for AI coding tool internals, the other is a focused resource for Claude AI customization.

## When to Choose system-prompts-and-models-of-ai-tools

Choose **system-prompts-and-models-of-ai-tools** if:

- You want a broad reference for understanding how multiple AI coding assistants (Cursor, Devin, Replit, etc.) work under the hood.
- You’re researching system prompts and internal tool architectures across different platforms.
- You prefer a well-established, highly starred repository (138k+ stars) with frequent community updates.
- You need a single source to compare prompts and models from 25+ tools.
- You’re a developer or AI enthusiast interested in the general landscape of AI coding assistants.

This tool is ideal for comparative analysis, learning prompt engineering patterns, or building your own AI coding tool by studying existing ones.

## When to Choose awesome-claude-skills

Choose **awesome-claude-skills** if:

- You are specifically working with Claude AI and want to customize its workflows.
- You need resources for automation, agent skills, or MCP (Model Context Protocol) integration.
- You prefer a curated, focused list rather than a broad aggregation.
- You’re building Claude-based agents or workflow automation tools.
- You want a community-driven collection that’s easier to navigate for Claude-specific tasks.

This tool is perfect for developers who are already committed to Claude AI and want to extend its capabilities with pre-built skills and tools.

## Verdict

Both **system-prompts-and-models-of-ai-tools** and **awesome-claude-skills** are excellent free resources, but they serve different needs.

- **Choose system-prompts-and-models-of-ai-tools** if you want a comprehensive, cross-platform reference for AI coding tool internals. Its massive star count and broad scope make it the go-to for understanding the AI coding assistant ecosystem.
- **Choose awesome-claude-skills** if you are focused on Claude AI customization, automation, and agent skills. It’s a more specialized, curated resource that’s easier to digest for Claude-specific projects.

For most developers, starting with **system-prompts-and-models-of-ai-tools** gives a broader foundation, while **awesome-claude-skills** is the better choice once you’ve committed to Claude AI. Both are free, so there’s no harm in exploring both to see which fits your workflow best.
