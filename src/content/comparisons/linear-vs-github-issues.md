## Pricing

### Linear
- **Free**: $0 — Up to 250 issues, Unlimited file uploads, Roadmap
- **Standard**: $8/mo — Unlimited issues, Guest accounts, SLA timers
- **Plus**: $14/mo — Multiple teams, Initiatives, Triage

### GitHub Issues
- **Free**: $0 — Unlimited public repos, 2,000 Actions minutes
- **Team**: $4/mo — Unlimited private repos, 3,000 Actions minutes, Code owners
- **Enterprise**: $21/mo — SAML SSO, Advanced auditing, Priority support
