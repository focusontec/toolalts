# Cursor vs VS Code

## Overview

Cursor is a fork of VS Code with deep AI integration, while VS Code is the extensible editor from Microsoft with the largest extension marketplace.

## Cursor

Cursor reimagines the editor experience with AI at its core: tab-based predictions, codebase chat, and inline AI edits.

## VS Code

VS Code offers the most mature ecosystem, thousands of extensions, and powerful debugging tools. With Copilot, it also has strong AI capabilities.

## Verdict

Choose **Cursor** if you want the most integrated AI experience.
Choose **VS Code** if you need the broadest extension ecosystem and the most mature tooling.
